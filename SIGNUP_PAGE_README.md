# Signup Page for Mobile Bookmarking

## Overview
The `/signup` page is a simplified, mobile-optimized version of the main site that focuses solely on the newsletter signup form. It's designed to be easily bookmarked on iPhone home screens.

## Features
- **Clean, focused design** - Just the header and signup form
- **Mobile-optimized** - Responsive design that works great on phones
- **Home screen icon** - Custom camping-themed favicon for iPhone bookmarks
- **Fast loading** - Minimal content for quick access

## How to Bookmark on iPhone

### Method 1: Safari Bookmark
1. Open Safari on your iPhone
2. Navigate to `yourdomain.com/signup`
3. Tap the Share button (square with arrow pointing up)
4. Select "Add to Home Screen"
5. Customize the name if desired (e.g., "Pack 338 Signup")
6. Tap "Add"

### Method 2: Direct Home Screen Addition
1. Open the signup page in Safari
2. Tap the Share button
3. Scroll down and tap "Add to Home Screen"
4. The page will appear as an app icon on your home screen

## What Happens When Bookmarked
- **App-like experience** - Opens in full-screen mode without Safari UI
- **Custom icon** - Shows the camping-themed favicon
- **Quick access** - One tap to open the signup form
- **Offline-friendly** - Basic functionality works without internet

## Technical Details
- **File**: `signup.html`
- **Layout**: Uses the same `default.html` layout as main site
- **Favicon**: `assets/images/favicon.svg` (SVG format for crisp display)
- **Mobile meta tags**: Optimized for iPhone home screen experience
- **Responsive CSS**: Automatically adjusts for different screen sizes

## Customization
The signup page inherits all the styling from the main site, so any changes to `assets/css/style.css` will automatically apply to this page as well.

## URL Structure
- **Main site**: `yourdomain.com/` or `yourdomain.com/index.html`
- **Signup page**: `yourdomain.com/signup` or `yourdomain.com/signup.html`

## Benefits for Users
- **Faster access** - No need to navigate through the main site
- **Mobile-optimized** - Form is sized perfectly for touch input
- **Professional appearance** - Looks like a native app when bookmarked
- **Easy sharing** - Parents can quickly share the signup link
